## Overview
This challenge is a Jeopardy-style Capture The Flag (CTF) conducted on a single Linux virtual machine.  
Participants are provided only with the VM image and must explore, enumerate, and exploit the system to discover hidden flags.

The system contains **10 flags**, each designed to test different Linux, enumeration, privilege escalation, and forensic skills.

No usernames or passwords are provided.  
All access must be obtained through analysis and exploitation of the system itself.

---

- Enumerate services, files, and permissions
- Escalate privileges where required
